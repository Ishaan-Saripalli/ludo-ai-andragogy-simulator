# ludo-ai-andragogy-simulator
A gamified educational simulator developed as part of a research study, applying andragogical learning principles to teach AI search algorithms (BFS, DFS, A*) through an interactive Ludo-based environment.

This is a research-driven, gamified simulation tool designed to teach fundamental Artificial Intelligence (AI) search algorithms through an interactive Ludo-based environment.

Developed as part of an academic study, this tool transforms abstract AI concepts into visual, intuitive, and experiential learning, making them accessible to students without a computer science background.

Objectives:
The primary goal of this project is to:
1. Gamify AI search algorithms
2. Promote conceptual understanding
3. Enable experiential and interactive learning
4. Support teaching AI to non-CS students

The simulator demonstrates three core AI search strategies (BFS, DFS and A* in red, blue and green colors respectively).\

Features:
1. Interactive simulation using pygame
2. Simultaneous execution of BFS, DFS, and A*
3. Real-time move logging panel
4. Auto-play mode
5. Strategy comparison in a controlled environment

Educational uses:
1. AI/ML introductory courses
2. Classroom demonstrations
3. Workshops & seminars
